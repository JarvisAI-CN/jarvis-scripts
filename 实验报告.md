### 题目1：计算圆的周长和面积

#### 3.1.1 算法设计

1. 输入圆的半径 r
2. 使用公式计算周长：C = 2πr
3. 使用公式计算面积：S = πr²
4. 输出周长和面积

#### 3.1.2 程序代码

```python
import math

def circle_calculation():
    """
    输入圆的半径，计算并输出圆的周长和面积
    """
    print("题目1：计算圆的周长和面积")
    print("="*50)

    try:
        radius = float(input("请输入圆的半径："))

        if radius <= 0:
            print("错误：半径必须为正数！")
            return

        # 计算周长和面积
        circumference = 2 * math.pi * radius
        area = math.pi * radius ** 2

        print(f"\n半径为 {radius} 的圆：")
        print(f"周长 = {circumference:.2f}")
        print(f"面积 = {area:.2f}")

    except ValueError:
        print("错误：请输入有效的数字！")
```

### 题目2：判断三位水仙花数

#### 3.2.1 算法设计

1. 输入一个三位数 n
2. 分解各位数字：百位a = n // 100，十位b = (n // 10) % 10，个位c = n % 10
3. 计算各位的立方和：sum = a³ + b³ + c³
4. 判断：若 n == sum，则是水仙花数，否则不是

#### 3.2.2 程序代码

```python
def narcissistic_number():
    """
    判断输入的三位数是否为水仙花数
    水仙花数：三位数abc = a³ + b³ + c³
    """
    print("题目2：判断三位水仙花数")
    print("="*50)

    try:
        num = int(input("请输入一个三位数："))

        if num < 100 or num > 999:
            print("错误：请输入100-999之间的三位数！")
            return

        # 分解各位数字
        hundreds = num // 100        # 百位
        tens = (num // 10) % 10      # 十位
        units = num % 10             # 个位

        # 计算各位的立方和
        sum_of_cubes = hundreds ** 3 + tens ** 3 + units ** 3

        print(f"\n{num} 的各位数字：")
        print(f"百位：{hundreds}")
        print(f"十位：{tens}")
        print(f"个位：{units}")
        print(f"\n{hundreds}³ + {tens}³ + {units}³ = {sum_of_cubes}")

        if num == sum_of_cubes:
            print(f"\n✓ {num} 是水仙花数！")
        else:
            print(f"\n✗ {num} 不是水仙花数！")

    except ValueError:
        print("错误：请输入有效的整数！")
```

### 题目3：判断闰年

#### 3.3.1 算法设计

1. 输入年份 year
2. 判断闰年条件：
   - 能被4整除但不能被100整除，或者
   - 能被400整除
3. 输出判断结果

#### 3.3.2 程序代码

```python
def leap_year():
    """
    判断输入的年份是否为闰年
    闰年规则：
    1. 能被4整除但不能被100整除，或者
    2. 能被400整除
    """
    print("题目3：判断闰年")
    print("="*50)

    try:
        year = int(input("请输入年份："))

        if year < 1:
            print("错误：年份必须大于0！")
            return

        # 判断闰年
        is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

        print(f"\n{year} 年：")
        if is_leap:
            print(f"✓ {year} 是闰年（366天）")
        else:
            print(f"✗ {year} 不是闰年（365天）")

    except ValueError:
        print("错误：请输入有效的整数！")
```

## 四、实验总结

### 4.1 实验收获

通过本次实验，我掌握了以下内容：

1. **Python基础语法**：熟练使用了变量、数据类型、运算符、表达式等基本语法
2. **输入输出操作**：掌握了`input()`和`print()`函数的使用方法，以及格式化输出
3. **条件判断**：学会了使用`if-else`语句进行逻辑判断
4. **数学运算**：使用了`math`模块中的数学常量（如`math.pi`）和运算符（如`**`幂运算）
5. **异常处理**：使用`try-except`捕获输入错误，提高程序的健壮性
6. **程序设计**：掌握了从算法设计到代码实现的完整流程

### 4.2 技术要点

1. **整数除法与取余**：
   - `//` 整数除法：用于分解数字的各位
   - `%` 取余运算：用于获取最后几位数字

2. **逻辑表达式**：
   - `and`、`or` 逻辑运算符的组合使用
   - 闰年判断：`(year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)`

3. **格式化输出**：
   - f-string：`f"周长 = {circumference:.2f}"`
   - 保留两位小数：`.2f`

### 4.3 遇到的问题及解决

1. **问题1**：输入非数字时程序报错
   - **解决**：使用`try-except`捕获`ValueError`异常

2. **问题2**：水仙花数数字分解容易出错
   - **解决**：使用整数除法`//`和取余`%`运算，分步分解各位数字

3. **问题3**：闰年判断规则复杂
   - **解决**：将判断条件分解为两个部分，用`or`连接

### 4.4 改进方向

1. 可以添加循环功能，让用户可以多次测试不同数据
2. 可以增加更多的输入验证，提高程序的健壮性
