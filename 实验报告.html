三、程序代码与运行结果

题目1：计算圆的周长和面积

程序代码：

import math

radius = float(input("请输入圆的半径："))
circumference = 2 * math.pi * radius
area = math.pi * radius ** 2
print(f"周长 = {circumference:.2f}")
print(f"面积 = {area:.2f}")
测试输入：半径 = 5.0
输出结果：
周长 = 31.42
面积 = 78.54

题目2：判断三位水仙花数

程序代码：

num = int(input("请输入一个三位数："))
hundreds = num // 100
tens = (num // 10) % 10
units = num % 10
sum_of_cubes = hundreds ** 3 + tens ** 3 + units ** 3

if num == sum_of_cubes:
    print(f"{num} 是水仙花数！")
else:
    print(f"{num} 不是水仙花数！")
测试结果：

123✗ 非水仙花数

题目3：判断闰年

程序代码：

year = int(input("请输入年份："))
is_leap = (year % 4 == 0 and year % 100 != 0) or (year % 400 == 0)

if is_leap:
    print(f"{year} 是闰年")
else:
    print(f"{year} 不是闰年")
测试结果：
年份判断结果说明
2000✓ 闰年能被400整除
2004✓ 闰年能被4整除且不能被100整除
2023✗ 平年不能被4整除
2024✓ 闰年能被4整除且不能被100整除
1900✗ 平年能被100整除但不能被400整除
2100✗ 平年能被100整除但不能被400整除
