## 🎯 操作步骤

### 1. 登录宝塔面板
- **URL**: http://82.157.20.7:8888/fs123456
- **操作**: 在VNC桌面Firefox中打开此URL

### 2. 添加站点
- 点击左侧 "网站"
- 点击 "添加站点"
- 填写信息：
  - 域名: `ceshi.dhmip.cn`
  - PHP版本: 8.3
  - 数据库: MySQL (创建同名数据库)
  - 用户名/密码: 自动生成
- 点击 "提交"

### 3. 上传代码
- 点击网站列表中的 "根目录" 链接
- 进入文件管理器
- 删除默认文件 (index.html等)
- 点击 "上传" 按钮
- 选择 `/var/www/html/expiry/` 下的所有PHP文件
- 上传完成

### 4. 配置数据库
- 点击左侧 "数据库"
- 找到 `ceshi_dhmip_cn` 数据库
- 点击 "管理"
- 进入phpMyAdmin
- 导入 `/var/www/html/expiry/database.sql`

### 5. 申请SSL证书
- 返回 "网站" 列表
- 点击 "设置" (齿轮图标)
- 选择 "SSL" 选项卡
- 选择 "Let's Encrypt"
- 勾选域名 `ceshi.dhmip.cn`
- 点击 "申请"
- 等待证书安装完成

### 6. 测试访问
- HTTP: http://ceshi.dhmip.cn
- HTTPS: https://ceshi.dhmip.cn
