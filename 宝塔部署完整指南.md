# 宝塔面板部署保质期系统 - 完整操作指南

**目标**: 在宝塔面板部署 https://github.com/JarvisAI-CN/expiry-management-system.git

---

## 🎛️ 方法1: 使用宝塔面板Git克隆（推荐）

### 步骤1: 登录宝塔面板
- 打开VNC桌面Firefox
- 访问: http://82.157.20.7:8888/fs123456
- 输入用户名和密码

### 步骤2: 添加站点
1. 点击左侧菜单 **"网站"**
2. 点击 **"添加站点"**
3. 填写信息：
   - **域名**: `ceshi.dhmip.cn`
   - **PHP版本**: 8.3
   - **数据库**: MySQL（创建同名数据库）
   - **用户名/密码**: 自动生成（请记录）
4. 点击 **"提交"**

### 步骤3: 克隆GitHub代码
1. 在网站列表中，点击 `ceshi.dhmip.cn` 的 **"根目录"**
2. 进入文件管理器
3. 点击顶部 **"远程下载"** 按钮
4. 选择 **"Git克隆"**
5. 填写：
   - **Git仓库**: `https://github.com/JarvisAI-CN/expiry-management-system.git`
   - **分支**: `main`（或留空默认）
   - **目录**: `/www/wwwroot/ceshi.dhmip.cn`
6. 点击 **"克隆"**
7. 等待克隆完成

### 步骤4: 设置文件权限
1. 在文件管理器中，全选所有文件
2. 点击 **"权限"** 按钮
3. 设置为 `755`（或 `644` 对于PHP文件）
4. 所有者设置为 `www-data`

### 步骤5: 申请SSL证书
1. 返回网站列表
2. 点击 `ceshi.dhmip.cn` 的 **"设置"**（齿轮图标）
3. 选择 **"SSL"** 选项卡
4. 选择 **"Let's Encrypt"**
5. 勾选 `ceshi.dhmip.cn`
6. 点击 **"申请"**
7. 等待证书安装完成

### 步骤6: 访问安装向导
- 打开新标签页访问: `http://ceshi.dhmip.cn/install.php`
- 填写数据库信息
- 设置管理员账号
- 完成安装

## 🎛️ 方法2: 使用SSH命令行

如果主人有SSH访问权限，提供这些命令：

```bash
# 1. 连接到宝塔服务器
ssh ubuntu@82.157.20.7

# 2. 进入网站目录
cd /www/wwwroot/

# 3. 克隆代码
sudo git clone https://github.com/JarvisAI-CN/expiry-management-system.git ceshi.dhmip.cn

# 4. 设置权限
cd ceshi.dhmip.cn
sudo chown -R www-data:www-data .
sudo chmod -R 755 .

# 5. 完成安装
# 在浏览器访问 http://ceshi.dhmip.cn/install.php
```
