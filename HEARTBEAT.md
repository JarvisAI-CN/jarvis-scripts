```json
}
```

1. 读取 `.current_task.json` 检查是否有未完成任务
2. 如果有 → **先恢复任务，再检查心跳**
3. 如果没有 → 正常执行心跳检查
4. 心跳检查时发现需要处理的任务 → 写入 `.current_task.json`
5. 完成任务后清除状态
