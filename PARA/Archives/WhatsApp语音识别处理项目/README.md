## 项目概述

实现WhatsApp语音消息的自动化识别与智能回复。当主人发送语音消息时，系统能够自动捕获、下载音频文件，使用智谱AI Whisper API转录为文字，并根据文字内容生成回复或执行指令。

**核心功能**:
- ✅ 自动语音识别（基于智谱AI SenseVoice）
- ✅ 智能回复生成（上下文感知）
- ✅ OpenClaw集成（无缝对接）
- ✅ 状态持久化（避免重复处理）
- ✅ 完整测试覆盖（单元+集成测试）

**技术栈**:
- Python 3.10+
- 智谱AI Whisper API
- FFmpeg（音频转换）
- OpenClaw消息框架
