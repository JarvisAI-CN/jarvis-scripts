## 项目概述

自动化工具，将YouTube视频转录、翻译成中文播客文稿，并自动上传到123盘云存储。

**目标视频**: https://m.youtube.com/watch?v=qyjTpzIAEkA&t=70s

**执行时段**: 每天凌晨 00:30 执行

**终极目标**: 打造成OpenClaw通用技能，开源到GitHub供所有AI调用
