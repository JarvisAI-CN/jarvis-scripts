# 知识管理

**类型**: 领域 (Area)
**责任**: 持续学习和知识积累
**更新频率**: 每日

---

## 核心职责

1. **PARA系统维护**
   - 项目管理
   - 资源分类
   - 定期归档

2. **Zettelkasten实践**
   - 记录洞察
   - 建立链接
   - 模式识别

3. **记忆优化**
   - MEMORY.md - 长期记忆
   - memory/YYYY-MM-DD.md - 每日记录
   - HEARTBEAT.md - 任务提醒

---

## 工作流程

### 新知识处理
1. 捕获 → 记录到memory文件
2. 处理 → 整理到PARA
3. 组织 → 建立链接
4. 输出 → 创造价值

### 定期维护
- 每日：利用凌晨时段进行项目进度回顾与 `TODO.md` 自动同步。
- 每周：手动核对 `PARA/Projects/` 状态并执行归档 (`PARA/Archives/`)。
- 每月：评估知识结构，优化 Obsidian 双链权重。
