# 图片资源目录

本目录用于存放系统的图片资源。

## 建议的子目录结构

- `logo/` - 系统Logo和品牌图片
- `icons/` - 图标和按钮图片
- `backgrounds/` - 背景图片
- `products/` - 商品相关图片
- `uploads/` - 用户上传的图片

## 图片格式建议

- Logo: SVG 或 PNG (透明背景)
- 图标: SVG 或 PNG (32x32, 64x64)
- 背景图片: JPG (压缩) 或 WebP
- 商品图片: JPG (压缩质量80%) 或 WebP

## 优化建议

1. 使用 WebP 格式以减少文件大小
2. 提供多种分辨率以支持响应式设计
3. 使用 lazy loading 延迟加载图片
4. 压缩图片以减少加载时间
