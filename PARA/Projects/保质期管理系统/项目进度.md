## 项目概述

保质期管理系统是一个专为企业或个人设计的库存管理工具，主要功能包括：
- 商品库存管理
- 保质期预警和提醒
- 扫码入库和出库
- 批量导入和导出
- 用户权限管理
- API 接口
- 数据统计和分析
