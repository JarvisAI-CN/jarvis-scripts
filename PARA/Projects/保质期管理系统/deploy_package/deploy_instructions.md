## 🚀 极简部署步骤

现在系统支持**全自动安装引导**，你不再需要手动创建数据库或导入 SQL。

### 第一步：上传文件

在宝塔终端中执行以下命令（或通过文件管理器上传）：

```bash
cd /www/wwwroot/ceshi.dhmip.cn
# 下载最新部署包
curl -s http://10.7.0.5:8888/index.php -o index.php
curl -s http://10.7.0.5:8888/db.php -o db.php
curl -s http://10.7.0.5:8888/install.php -o install.php
# 设置权限
chmod 644 *.php
chown www:www *.php
# 清理旧默认页
rm -f index.html
```

### 第二步：运行安装引导

1. **访问你的域名**: `http://ceshi.dhmip.cn`
2. **自动跳转**: 系统会自动跳转到 `install.php` 安装引导页。
3. **填写信息**:
   - **数据库配置**: 输入你的数据库账号密码。
   - **管理员设置**: 设置你的首个管理员账号和密码。
4. **点击安装**: 剩下的工作（创建数据库、建表、初始化）全部由系统自动完成！
