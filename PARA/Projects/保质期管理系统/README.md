# 保质期管理系统

**创建时间**: 2026-02-15
**状态**: 🔄 进行中 (v2.1 分类管理与健康看板版)
**优先级**: ⭐⭐⭐⭐⭐ (最高)

---

## 项目概述

从基础的“扫码录入工具”进化为“智能化主动安全平台”。核心目标：通过 AI 和自动化手段，彻底杜绝过期商品上架。

---

## 🎯 发展路线图 (Roadmap)

### v1.0.x - 基础奠基 (已完成)
- [x] 手机浏览器扫码功能
- [x] 多批次管理逻辑

### v1.1.x - 体验优化 (已完成)
- [x] AI 自动排序报表 (找到期的排在前面)
- [x] 扫码成功提示音 ("叮"声)

### v1.2.x - 系统化升级 (已完成)
- [x] 自动化安装引导 (install.php)
- [x] 用户权限控制 (前台登录)
- [x] 后台一键热更新 (在线升级)
- [x] 自定义 AI API 配置

### v2.1.x - 分类管理与健康看板 (研发中)
- [x] **分类管理**: 酸奶/饼干(小食品)、物料、咖啡豆，支持不同下架规则
- [x] **提前下架**: 支持设置每个商品提前 N 天提醒/下架
- [x] **数据可视化大屏**: 首页展示“效期健康度”仪表盘
- [x] **操作日志**: 记录关键操作（保存、更新等）
- [x] **主动预警中心**:
    - [x] 每日凌晨自动巡检数据库 (check_expiry.py)。
    - [x] 过期前 3/7/15 天自动发送预警邮件 (已集成 email_tool.py)。
- [ ] **AI OCR 识别**: 扫描包装上的“生产日期/保质期”文本，AI 自动解析。
- [ ] **AI 销售辅助**: 针对临期商品，AI 自动生成对应的促销文案或打折策略。
