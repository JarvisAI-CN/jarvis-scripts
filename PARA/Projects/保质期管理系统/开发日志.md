**项目名称**: 保质期管理系统
**项目状态**: 🟢 进行中
**当前版本**: v2.13.1
**技术栈**: PHP + MySQL + Bootstrap 5

---

## 项目概述

保质期管理系统是一个专为企业或个人设计的库存管理工具，主要功能包括：
- 商品库存管理
- 保质期预警和提醒
- 扫码入库和出库
- 批量导入和导出
- 用户权限管理
- API 接口
- 数据统计和分析

## 技术架构

### 后端架构
- PHP 7.4+
- MySQL 5.7+
- 单页应用架构
- RESTful API

### 前端架构
- Bootstrap 5
- jQuery
- HTML5 QRCode 库
- 响应式设计

### 核心文件
- `index.php`: 主页面
- `admin.php`: 后台管理
- `api.php`: API 接口
- `db.php`: 数据库连接
- `database.sql`: 数据库结构

## 部署信息

### 服务器配置
- 操作系统: Linux (CentOS 7 或 Ubuntu)
- 服务器软件: Nginx 或 Apache
- PHP 版本: 7.4+
- MySQL 版本: 5.7+

### 部署步骤
1. 上传文件到服务器
2. 配置数据库连接
3. 导入数据库结构
4. 配置虚拟主机
5. 访问网站进行安装

## 故障排查

### 常见问题

1. **MySQL 连接失败**
   - 检查数据库连接配置
   - 检查数据库服务是否运行
   - 检查防火墙设置

2. **扫码功能无法使用**
   - 检查摄像头权限
   - 检查网络连接
   - 检查浏览器兼容性

3. **API 接口报错**
   - 检查请求参数
   - 检查权限设置
   - 检查数据库连接

## 开发规范

### 代码规范
- 使用 UTF-8 编码
- 遵循 PSR-4 规范
- 使用 4 空格缩进
- 函数和变量使用蛇形命名法

### 安全规范
- 使用预处理语句防止 SQL 注入
- 对用户输入进行验证和过滤
- 对敏感信息进行加密存储

### 测试规范
- 每个功能都需要进行测试
- 测试代码覆盖核心功能
- 使用单元测试框架进行自动化测试
