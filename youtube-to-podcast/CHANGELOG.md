## [0.1.0] - 2026-02-08

### Added
- Initial release
- YouTube audio download (yt-dlp)
- Whisper API transcription
- GPT-4 translation to podcast-style Chinese
- WebDAV/123 Pan cloud upload
- Crontab automation support
- Comprehensive documentation
- OpenClaw skill packaging

### Features
- Automatic pipeline: download → transcribe → translate → upload
- Natural spoken Chinese translation
- Podcast-style formatting with voice cues
- Conversational intro/outro
- Error handling and logging
- Troubleshooting guide
