## 💡 安装方式

### ClawHub CLI（推荐）
```bash
npx clawhub@latest install <skill-slug>
```

### 手动安装
1. **全局**: `~/.openclaw/skills/`
2. **工作区**: `<project>/skills/`
3. **优先级**: Workspace > Local > Bundled

### 替代方式
直接粘贴GitHub仓库链接到聊天，让助手自动处理。
