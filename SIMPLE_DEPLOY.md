## 🎯 方案1：宝塔文件管理器上传（最简单）

### 步骤1：打开宝塔文件管理器

1. **登录宝塔**: http://82.157.20.7:8888/fs123456
2. **点击左侧**: 网站
3. **找到**: ceshi.dhmip.cn
4. **点击**: "根目录"

### 步骤2：上传2个PHP文件

在宝塔文件管理器中：

1. **点击右上角**: "上传"按钮
2. **选择文件**: 一次性选择这2个文件
   ```
   /home/ubuntu/.openclaw/workspace/PARA/Projects/保质期管理系统/deploy_package/index.php
   /home/ubuntu/.openclaw/workspace/PARA/Projects/保质期管理系统/deploy_package/db.php
   ```

3. **等待上传完成**（约10秒）

4. **确认**: 文件列表中应该有:
   - index.php (46KB)
   - db.php (2.5KB)

### 步骤3：删除默认文件（如果有）

如果文件列表中有 `index.html`:
- 右键点击 → 删除

### 步骤4：测试访问

打开浏览器访问：
```
http://ceshi.dhmip.cn
```

应该看到"保质期管理系统"界面。

## 🎯 方案2：通过宝塔终端上传（备选）

如果文件管理器上传不方便：

1. **在宝塔面板中**: 左侧菜单 → 终端
2. **复制粘贴以下命令**:

```bash
# 进入网站目录
cd /www/wwwroot/ceshi.dhmip.cn

# 下载文件（使用临时文件服务器）
curl http://10.7.0.5:8888/index.php -o index.php
curl http://10.7.0.5:8888/db.php -o db.php

# 设置权限
chmod 644 *.php
chown www:www *.php

# 验证
ls -lh
```

3. **回车执行**
