| metal2 | 第二层金属 | 紫色 |

### 3.2 版图设计步骤

#### 步骤1：创建版图视图
```
Cell: inv → File → New → Cell View
- View Name: layout
- Tool: Virtuoso
```

#### 步骤2：设置栅格和显示
```
Options → Display
- Grid: 0.005 (最小栅格)
- Major Grid: 0.05
```

#### 步骤3：绘制PMOS（上方）
```
1. nwell层:
   - 尺寸: 从(0, 1.0)到(3.0, 3.0)
   - 宽度: 3.0 μm
   - 高度: 2.0 μm

2. active层:
   - 尺寸: 从(0.5, 1.5)到(2.5, 2.5)
   - 宽度: 2.0 μm
   - 高度: 1.0 μm

3. pselect层:
   - 包围active层
   - 重叠: 0.3 μm

4. poly层（栅极）:
   - 位置: x = 1.5
   - 宽度: 0.18 μm
   - 延伸active外: 0.2 μm

5. contacts (接触孔):
   - 源极接触: (1.0, 2.0)
   - 漏极接触: (2.0, 2.0)
   - 尺寸: 0.22 x 0.22 μm
```

#### 步骤4：绘制NMOS（下方）
```
1. active层:
   - 尺寸: 从(0.5, 0.0)到(2.0, 1.0)
   - 宽度: 1.5 μm (600nm * 2.5用于接触)
   - 高度: 1.0 μm

2. nselect层:
   - 包围active层
   - 重叠: 0.3 μm

3. poly层（栅极）:
   - 与PMOS对齐
   - 位置: x = 1.5
   - 宽度: 0.18 μm

4. contacts (接触孔):
   - 源极接触: (1.0, 0.5)
   - 漏极接触: (2.0, 0.5)
```

#### 步骤5：绘制互连
```
1. metal1 (水平连接):
   - vdd连线: y = 2.5, 宽度0.3μm
   - vss连线: y = 0.0, 宽度0.3μm
   - 输出Y: 垂直连接两管漏极

2. via1:
   - 在需要连接metal1和metal2处

3. metal2 (电源和地):
   - vdd: 顶部水平走线
   - vss: 底部水平走线
```

#### 步骤6：引出端口
```
1. Pin层绘制:
   - vdd: 顶部，metal2
   - vss: 底部，metal2
   - A:   左侧，poly
   - Y:   中间，metal1/metal2

2. 添加Pin属性:
   - 选中Pin形状
   - Add Pin → 设置名称和类型
```

## 五、常见问题排查

### 问题1：DRC出现间距错误
**原因**：金属或多晶间距小于规则
**解决**：
```
- 增加金属宽度或间距到0.3μm
- 确保最小间距满足设计规则
```

### 问题2：LVS显示端口不匹配
**原因**：版图端口名称或类型与原理图不一致
**解决**：
```
1. 检查原理图端口名称（A, Y, vdd, vss）
2. 检查版图Pin层名称完全匹配
3. 确认端口类型（Input, Output, InputOutput）
```

### 问题3：LVS显示器件数量不匹配
**原因**：版图中器件数量或参数与原理图不一致
**解决**：
```
1. 检查PMOS和NMOS的W和L参数
2. 确认m=1（无并联）
3. 检查Device Recognition是否正确识别器件
```

### 问题4：接触孔未连接
**原因**：contact层与metal或active未正确重叠
**解决**：
```
- 确保contact完全包围在metal和active中
- 重叠至少0.05μm
```
