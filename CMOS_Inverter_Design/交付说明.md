## ⚠️ 重要注意事项

### 1. 工艺库选择
- 确保使用180nm工艺库（如tsmc18）
- 晶体管模型名称必须是 `p18` 和 `n18`
- 不同EDA工具的模型名可能不同，需调整

### 2. 尺寸单位
- Virtuoso中默认单位通常是微米 (μm)
- 输入时注意单位转换：
  - `1u` = 1 μm
  - `180n` = 0.18 μm
  - `600n` = 0.6 μm

### 3. 端口类型
- **vdd/vss** 必须是 `inputOutput` 类型
- **A** 必须是 `input` 类型
- **Y** 必须是 `output` 类型
- 类型不匹配会导致LVS失败

### 4. 版图层顺序
- 确保层的正确叠加关系
- nwell在最底层，metal2在最上层
- 接触孔必须完全被金属包围

### 5. 验证顺序
- 先完成DRC验证
- DRC通过后再运行LVS
- LVS需要同时有schematic和layout

## 🛠️ 常见问题排查

### Q1: DRC显示大量错误
**原因**：设计规则不满足
**解决**：
1. 检查金属宽度是否≥0.3μm
2. 检查间距是否满足最小间距
3. 使用stretch工具调整尺寸
4. 查看具体错误信息逐一修改

### Q2: LVS显示端口不匹配
**原因**：端口名称或类型不一致
**解决**：
1. 在schematic中检查端口名称
2. 在layout中检查pin名称
3. 确保完全匹配（区分大小写）
4. 检查端口类型

### Q3: LVS显示器件数量不匹配
**原因**：版图器件未被正确识别
**解决**：
1. Verify → Markers → Delete All
2. Verify → Extract 重新提取
3. 检查器件参数W和L
4. 重新运行LVS

### Q4: 无法找到p18或n18模型
**原因**：工艺库配置问题
**解决**：
1. 检查库是否正确附加
2. 在Component面板Browse中查找
3. 可能需要使用其他模型名
4. 联系老师确认工艺库
